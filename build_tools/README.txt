马里奥手势控制游戏 (Mario Hand Gesture Game)
版本: 1.0.0

========================================
项目简介
========================================

这是一个使用手势控制马里奥游戏的Python项目。
通过摄像头检测手势，实时控制马里奥的移动和跳跃。

========================================
系统要求
========================================

- Windows 10/11 (64位)
- 摄像头（内置或外置）
- 至少4GB可用内存
- 至少3GB可用磁盘空间

========================================
功能特性
========================================

1. 手势识别控制
   - 张开手掌：跳跃
   - 握拳：奔跑

2. 区域控制
   - 左侧区域：向左移动
   - 中间区域：停止
   - 右侧区域：向右移动

3. 智能摄像头适配
   - 自动搜索可用摄像头
   - 智能选择最佳后端
   - 自动检测最佳分辨率

========================================
安装方法
========================================

方式1：使用安装程序（推荐）
1. 双击 MarioHandGesture_Setup.exe
2. 选择安装路径
3. 点击"安装"
4. 安装完成后自动启动

方式2：直接运行exe
1. 进入dist目录
2. 双击 MarioHandGesture.exe
3. 等待摄像头初始化

========================================
使用说明
========================================

1. 启动游戏后会自动打开摄像头窗口
2. 马里奥游戏会在全屏模式下运行
3. 站在摄像头前，确保光线充足
4. 使用手势控制马里奥

控制方式：
- 左侧+张开：向左跳跃
- 左侧+握拳：向左跑
- 中间+张开：跳跃
- 中间+握拳：原地不动
- 右侧+张开：向右跳跃
- 右侧+握拳：向右跑

按 'q' 键退出游戏

========================================
故障排除
========================================

问题1：摄像头无法打开
解决：
- 检查摄像头是否连接
- 关闭其他占用摄像头的应用
- 尝试手动指定摄像头：--camera_id 1

问题2：游戏画面不显示
解决：
- 检查是否有杀毒软件拦截
- 等待几秒让游戏初始化
- 重启程序

问题3：手势识别不准确
解决：
- 确保环境光线充足
- 站在摄像头正前方
- 手势幅度要明显
